Motive

BasicsTech’s vision is to help spread latest software technologies in tier 2 and tier 3 cities, starting with Amravati.


This will enable local youth to develop world class software solutions. So that they become highly employable, create their own products and should they wish, they could stay in their town/city and start something on their own or work for a local firm.

Previoiusly, there was big dilemna to students about which course/technology to choose. For that they used to go to tier 1 cities, spend a lot of money on classes and still were not able to find jobs. But with BasicsTech's latest technologies at student's doorstep, they can avoid big this trap.

We have expertise in latest tools and technologies such as:
  • Advanced javascript
  • Reactjs
  • Angularjs
  • Nodejs
  • Ionic
  • Mongodb - The NoSQL database
  • Flux/Redux
  • JQuery
  • ES6/ES7/Babel
  • Expressjs
  • React Native
  • Phonegap/Cordova
  • Grunt/Gulp
  • HTML5
  • CSS3
  • Web Components
  • Polymer
Along with development of Websites, Webapps, Native apps, and Server side solutions, we will be introducing hardware/IOT programming using tools, frameworks and resources like:
  • Johny Five
  • Cylonjs
  • Node Copter
  • Node Bots
  • ARDrones
  • Arduino

And if all of above sounds too foreign and intimidating to you. Fear not! We are fun loving people! We think learning should be fun and we teach that way :) . You will be learning all of these while building software products that actually address real life needs and problems. It’s going to be all practical, hands on, no-nonsense and fun way of learning and teaching!

Javascript (Please note Javascript and Java are two very different languages) is world's no. 1 and very powerful language. We will be using a lot of it to create delightful software.

We are looking to reach to as many students(B.E, polytechnic, BCA, MCA, Bsc, Msc or anybody interested irrespective of branch/education) and professionals as possible. Also we are looking to have tie ups with colleges and local companies for training. Please feel free to contact us!

So let’s embark on a fun filled journey to create wonderful, real problem solving software. Come join us! Lets rock!! :)

Note:

1) Even though these technologies are advanced and powerful. They are easy to learn. It's like learning bike compared to horse!

2) We teach in such a way that they become even more easier.
